## Table of Contents

## Krav
npm `^3.0.0`
node `^6.9.5`

## Installering

$ git clone https://github.com/andreasnc/summer-project-tasks-2017.git <my-project-name>
$ cd <my-project-name>

$ npm install

## Kjør prosjektet

Etter at steget [Kjør prosjektet](#running-the-project) er komplett, så kan prosjektet kjøres:

$ npm start
